# Promptly

**The Ultimate AI API Testing Tool**

Test Claude, GPT, Gemini, Mistral, and Grok APIs in one powerful desktop application.

[![Website](https://img.shields.io/badge/Website-promptly.mint--soft.com-blue)](https://promptly.mint-soft.com/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Version](https://img.shields.io/badge/Version-1.0.0-orange.svg)](https://github.com/mintsoft-app-series/promptly-app/releases)

## Features

- **Multi-Provider Support** - Access Anthropic, OpenAI, Google, Mistral, and Grok (X.AI) APIs from a single interface
- **Real-time Cost Calculation** - Token counting and cost estimation before sending requests
- **Secure Storage** - API keys are encrypted and stored locally. Your data never leaves your machine
- **History & Templates** - Save request history and create reusable prompt templates
- **Cross-Platform** - Available for Windows, macOS, and Linux
- **Dark Mode** - Built-in dark mode for comfortable use

## Supported AI Models

| Provider | Models |
|----------|--------|
| Anthropic | Claude Opus 4.5, Claude Sonnet 4, Claude Haiku 3.5 |
| OpenAI | GPT-4o, GPT-4o-mini, GPT-4-turbo, GPT-3.5-turbo |
| Google | Gemini 2.0 Flash, Gemini 1.5 Pro, Gemini 1.5 Flash |
| Mistral | Mistral Large, Mistral Medium, Mistral Small |
| Grok (X.AI) | Grok 2, Grok 2 mini |

### macOS Installation Note

If you encounter a security warning when opening the app, run the following command in Terminal:

```bash
sudo xattr -rd com.apple.quarantine /Applications/Promptly.app
```
